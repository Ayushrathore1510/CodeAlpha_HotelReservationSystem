# CodeAlpha_HotelReservationSystem

## Description

Hotel Reservation System is a Java-based console application developed as part of the CodeAlpha Java Programming Internship. The system allows users to view available rooms, book rooms, cancel reservations, simulate payments, and manage booking records using File I/O.

## Features

* View available hotel rooms
* Room categorization (Standard, Deluxe, Suite)
* Book hotel rooms
* Cancel reservations
* Payment simulation
* View booking details
* Store booking records in a file (bookings.txt)
* Console-based user interface

## Technologies Used

* Java
* File I/O
* Object-Oriented Programming (OOP)

1. Compile the program:
